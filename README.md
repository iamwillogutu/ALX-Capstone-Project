# Currency Converter Application

## Project Description

A simple and responsive web application that allows users to convert currencies using live exchange rates from a public API. Built with React and styled with Tailwind CSS, this app provides an easy way to convert between multiple currencies with real-time results.

---

## Features

* Convert amounts between different currencies
* Select source and target currencies from dropdown menus
* Swap currencies with a single click
* Real-time conversion using live API rates
* Responsive design using Tailwind CSS
* Error handling for invalid inputs and API errors

## Installation

1. Clone the repository:

   ```
   git clone https://github.com/yourusername/currency-converter.git
   ```
2. Navigate to the project folder:

   ```
   cd currency-converter
   ```
3. Install dependencies:

   ```
   npm install
   ```
4. Start the development server:

   ```
   npm run dev
   ```

---

## Usage

1. Enter the amount to convert.
2. Select the source and target currencies.
3. Click 'Convert' or swap currencies if needed.
4. View the converted amount instantly.

---

## API

This app uses [ExchangeRate-API](https://www.exchangerate-api.com/) to fetch live currency exchange rates.
